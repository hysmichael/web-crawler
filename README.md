# web-crawler
Python web crawler for Madewell, TopShop products and Weibo profile images.

### usage

Dependencies:
```
beautifulsoup4==4.3.2
lxml==3.4.4
ndg-httpsclient==0.4.0
pyOpenSSL==0.15.1
pyasn1==0.1.7
requests==2.7.0
rsa==3.1.4
six==1.9.0
wsgiref==0.1.2
```

Crawl new arrivals for Madewell.
```
$ python madewell.py
```

Crawl new arrivals for TopShop.
```
$ python topshop.py
```
